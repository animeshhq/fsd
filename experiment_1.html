Full Stack Developer Curriculum

The Full Stack Development program covers both frontend and backend technologies. It emphasizes structured learning, databases, and deployment practices.

Course Objectives

  1. Master HTML, CSS, and JavaScript basics.
  2. Develop server-side applications using Node.js.
  3. Understand databases (SQL & NoSQL).
  4. Deploy applications to the cloud.

Modules

1. Frontend Basics

  • HTML: headings, paragraphs, lists, & formatting
  • CSS: box model, selectors, layout
  • JavaScript: variables, functions, DOM manipulation

2. Backend Development

Node.js
Building APIs with Express framework
Databases
SQL (MySQL) & NoSQL (MongoDB)

3. Deployment & Tools

  • Version control with Git
  • Hosting on GitHub Pages and Cloud platforms
  • Introduction to CI/CD pipelines

4. Summary of this course

  1. Frontend
    • HTML
    • CSS
  2. Backend
    • Node.js
    • Databases
      1. SQL
      2. NoSQL

Capstone Project

Build a full-stack application integrating frontend, backend, and deployment concepts.
